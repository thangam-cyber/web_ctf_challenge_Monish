MODULE 01 — RETINAL STROBE (Kanna moodi thirakkanum!)
Enna nadakkuthu: Oru TV static noise aah irukkum. Athula thideer nu
oru pink/magenta color box varum, just 1 frame (romba fast ah!).
Solve panrathu epdi: Antha pink box enga varutho, atha pathu udane
correct ah 0.5 seconds kulla click pannanum.
AI yen fail aagum: AI aala kanna vechu paka mudiyathu. Code la antha
box entha edathula varum nu ezhuthave illa (random position at runtime).

MODULE 02 — TEMPORAL WATERMARK (Browser inspection thiranaai!)
Enna nadakkuthu: Innoru noise image. Ana athula 4 secret pixels irukku.
Antha 4 pixels oda color: Green=6, Blue=28 irukkum.
Solve panrathu epdi: Pasanga console la poi JS code ezhuthi antha
(G=6, B=28) irukka pixels ah thedi kandupudikkanum. Kandupudicha 4
digits kedaikum: 7, 3, 5, 1. Input box la "7351" nu enter pannanum.
AI yen fail aagum: Antha pixels enga irukkum nu formula (sin/cos)
vechu mathitikitte irukkum (every 250ms). AI aala live ah canvas ah
scan panna JS ezhuthi run panna kashtam.

MODULE 03 — ACOUSTIC GHOST (Kaadhu kuduthu kelkanum!)

AI yen fail aagum: Text-padikkira bot ku kaadhu kekkathu. Athu antha

MODULE 04 — PRESSURE SIGNATURE (Kaiala draw pannanum!)

touch aalaiyo draw pannanum.
